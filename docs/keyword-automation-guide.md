# คู่มือการตั้งค่า Keyword Automation (Comment-to-DM)

ระบบ Keyword Automation จะช่วยให้คุณตอบกลับลูกค้าที่คอมเมนต์บนหน้าเพจ Facebook หรือ Instagram ได้โดยอัตโนมัติ พร้อมส่งรายละเอียดเชิงลึกเข้า Inbox ทันทีที่มีการพิมพ์คำค้นหาที่คุณตั้งไว้

---

## 🏷️ ตัวแปร (Dynamic Tags) ที่สามารถใช้งานได้

คุณสามารถใช้ "แท็ก" เหล่านี้ในเนื้อหาข้อความเพื่อดึงข้อมูลจากระบบมาใส่ได้อัตโนมัติ:

| Tag                 | Description                   | Example Output                         |
| :------------------ | :---------------------------- | :------------------------------------- |
| `{{title}}`         | Property title                | "Metro Sky บางซ่อน Interchange"        |
| `{{price}}`         | Current price (sale/rent)     | "฿2,390,000" or "฿10,000 / เดือน"      |
| `{{original}}`      | Original price (for discount) | "฿2,600,000"                           |
| `{{description}}`   | Full description              | (Text block)                           |
| `{{link}}`          | Website link                  | "https://your-site.com/properties/..." |
| `{{bedrooms}}`      | Number of bedrooms            | "1"                                    |
| `{{bathrooms}}`     | Number of bathrooms           | "1"                                    |
| `{{size_sqm}}`      | Size in sqm                   | "30"                                   |
| `{{floor}}`         | Floor level                   | "22"                                   |
| `{{property_type}}` | Property Type                 | "CONDO", "HOUSE", etc.                 |
| `{{listing_type}}`  | Listing Type                  | "ขาย", "ให้เช่า"                       |
| `{{location}}`      | District and Province         | "บางซื่อ กรุงเทพมหานคร"                |
| `{{transit}}`       | Nearest transit               | "MRT บางซ่อน (350 ม.)"                 |
| `{{google_maps}}`   | Google Maps URL               | "https://maps.google.com/..."          |
| `{{agent_name}}`    | Assigned Agent Name           | "K. Hunter"                            |
| `{{agent_phone}}`   | Agent Phone                   | "093-550-2143"                         |
| `{{agent_line}}`    | Agent LINE ID                 | "hunterz_rocker"                       |

## 📝 ตัวอย่างการตั้งค่า (Use Cases)

### 1. เน้นให้ข้อมูลและราคา (สำหรับลูกค้าทั่วไป)

- **Keyword**: `สนใจ`, `ราคา`, `เท่าไหร่`
- **DM Message**:
  > สวัสดีครับคุณลูกค้า ขอบคุณที่สนใจโครงการ **{{title}}** ครับ
  >
  > 💰 ราคาพิเศษ: **{{price}}** (จากเดิม {{original}})
  > 📖 รายละเอียด: {{description}}
  >
  > สามารถดูรูปเพิ่มเติมและแผนที่ได้ที่นี่ครับ: {{link}}
  >
  > หากต้องการนัดชมห้อง พิมพ์ "นัด" เพื่อคุยกับเจ้าหน้าที่ได้เลยครับ!

### 2. เน้นเร่งการตัดสินใจ (Flash Sale / ราคาพิเศษ)

- **Keyword**: `โปร`, `ลดได้ไหม`, `จอง`
- **DM Message**:
  > **[โปรโมชั่นพิเศษ!]** ทรัพย์สิน: {{title}}
  > ลดราคาจาก {{original}} เหลือเพียง **{{price}}** เท่านั้น!
  >
  > ดูข้อมูลยืนยันราคา: {{link}}
  >
  > ราคานี้ถึงสิ้นเดือนเท่านั้นนะครับ สนใจรับสิทธิ์พิมพ์ "รับสิทธิ์" ได้เลยครับ

### 3. สำหรับทรัพย์สินเช่า (Rental Inquiries)

- **Keyword**: `เช่า`, `ว่างไหม`, `เดือนเท่าไหร่`
- **DM Message**:
  > ยินดีครับ! **{{title}}** ยังว่างให้เช่าอยู่นะครับ
  >
  > 🔑 ค่าเช่าเพียง **{{price}}** ต่อเดือน
  > ✨ จุดเด่น: {{description}}
  >
  > นัดดูห้องจริงได้ทุกวันครับ ดูลิงก์นี้ประกอบการตัดสินใจ: {{link}}

## 💡 เทคนิคการตั้งค่าให้ได้ผลดีเยี่ยม

1.  **ตั้งคำให้ครอบคลุม**: ตรวจสอบคำที่ลูกค้ามักพิมพ์ผิด เช่น `สนจา`, `ขอราคา`
2.  **ใช้ Public Reply คู่กัน**: ตั้งค่าคำตอบกลับสาธารณะ (Public Reply) เช่น _"ส่งข้อมูลให้ทางแชทเรียบร้อยแล้วนะครับคุณลูกค้า 😊"_ เพื่อให้คนอื่นเห็นว่าคุณตอบไว
3.  **ใส่ Link เสมอ**: เพื่อให้ลูกค้าย้อนกลับมาดูข้อมูลบนเว็บและเก็บเป็น Lead ต่อไปได้
4.  **รูปภาพมาอัตโนมัติ**: หากทรัพท์สินมีรูป ระบบจะส่ง **Photo Album (Carousel)** ตามไปหลังจากข้อความ DM เสมอ ไม่ต้องตั้งค่าเพิ่มครับ
