# 👤 คู่มือการใช้งานสำหรับผู้ใช้ (User Manual)

วิธีใช้งานฟีเจอร์ต่างๆ ในระบบ Real Estate CRM สำหรับ Agent และ Admin

> **อัปเดตล่าสุด:** 27 กุมภาพันธ์ 2026 (Enterprise Multi-Tenant Edition)

---

## 1. การจัดการทรัพย์สิน (Property Management)

- **เพิ่มทรัพย์สินใหม่:** กรอกข้อมูลเบื้องต้น ระบบจะมีปุ่ม **"AI Generator"** ช่วยเขียนคำบรรยายให้น่าสนใจ 3 ภาษา (ไทย, อังกฤษ, จีน) อัตโนมัติ
- **การจัดการรูปภาพ:** ลากวางรูปเพื่ออัพโหลด และสามารถเลือกรูป "Cover" ได้เพียงคลิกเดียว
- **สถานะทรัพย์สิน:** ปรับเปลี่ยนสถานะ (Active, Sold, Rented) เพื่อให้ระบบนำออกจากหน้าเว็บสาธารณะทันที
- **Global Inventory:** สำหรับ Admin คลังทรัพย์สินรวมที่ดึงข้อมูลจากทุกสาขามาแสดงผลในหน้าเดียว (หน้าตาแบบ Premium Glassmorphism)
- **ถังขยะ (Trash):** ทรัพย์ที่ถูกลบจะเก็บไว้ในถังขยะ สามารถกู้คืนได้ ระบบจะลบถาวรอัตโนมัติผ่าน Cron Job

---

## 2. ระบบ AI Chatbot อัจฉริยะ (AI Search Assistant)

ลูกค้าสามารถพิมพ์คุยกับ Chatbot หน้าเว็บไซต์เพื่อหาบ้านที่ต้องการ:

- **Natural Language Search:** พิมพ์เช่น "หาคอนโดใกล้ BTS ราคากลางๆ" AI จะคำนวณและแสดงผลที่ตรงที่สุด
- **Lead Capture:** หากลูกค้าสนใจ AI จะขอชื่อและเบอร์ติดต่อ/LINE ID เพื่อสร้าง Lead เข้าสู่ระบบ CRM ให้อัตโนมัติ
- **Smart Matching:** ระบบจะคำนวณ Match Score ระหว่างความต้องการลูกค้ากับทรัพย์สินที่มีในระบบ

---

## 3. ระบบตอบกลับรวม (Omni-channel Inbox)

จัดการการพูดคุยกับลูกค้าทุกช่องทางในหน้าเดียว:

- **Centralized Chat:** ข้อความจาก LINE OA, Facebook Messenger, Instagram DM, WhatsApp จะมารวมกันที่หน้า Dashboard
- **Direct Reply:** Agent สามารถพิมพ์ตอบโต้ลูกค้าได้โดยตรงจากระบบ CRM ไม่ต้องสลับแอป
- **Real-time Notifications 🔔:** เมื่อมีลูกค้าทักมาใหม่ หรือมีการโอนย้ายงาน ระบบจะส่งแจ้งเตือนเด้งขึ้นที่ **Notification Bell** (มุมขวาบน) ทันทีไม่ต้อง Refresh
- **Platform Badge:** แต่ละข้อความจะแสดง Icon ของช่องทาง (LINE/FB/IG/WA) เพื่อให้ Agent ทราบแหล่งที่มา

## 4. ระบบโพสต์ลง Social Media 🆕

โพสต์ข้อมูลทรัพย์สินไปยัง Social Media ได้โดยตรงจากระบบ CRM:

### Facebook & Instagram

- ไปที่หน้ารายการทรัพย์สิน → คลิกเมนู **"⋯"** ข้างทรัพย์ → เลือก **"โพสต์ลง Facebook"** หรือ **"โพสต์ลง Instagram"**
- ระบบจะสร้างเนื้อหาตาม Template ที่ตั้งค่าไว้ พร้อมอัพโหลดรูปภาพอัตโนมัติ
- ติดตามสถานะการโพสต์ผ่าน **Social Post Monitor** (แถบด้านล่างขวาจอ)

### TikTok

- ต้องเชื่อมต่อบัญชี TikTok ก่อนที่หน้า **Settings > Social**
- เลือกทรัพย์สิน → คลิก **"โพสต์ลง TikTok"** → ใส่ Caption → ส่งโพสต์

### Keyword Automation (Comment-to-DM)

- ตั้งค่าที่หน้า **Settings > Social > Keyword Automation**
- กำหนดคำค้นหา (เช่น "สนใจ", "ราคา") → เมื่อลูกค้าคอมเมนต์คำเหล่านี้บนโพสต์ → ระบบจะส่ง DM ข้อมูลทรัพย์อัตโนมัติ
- รองรับทั้ง Facebook และ Instagram Comments

## 5. การจัดการบทความ (Blog System)

1.  ไปที่เมนู **"Blogs"**
2.  ใช้ฟีเจอร์ **"AI Blog Writer"** เพื่อช่วยคิดหัวข้อและเนื้อหาที่น่าสนใจ (เช่น "5 เทคนิคเลือกคอนโดใกล้รถไฟฟ้า")
3.  การลงบทความบ่อยๆ จะช่วยให้เว็บไซต์ติดอันดับ Google (SEO) ได้ดีขึ้น

---

## 6. ระบบแจ้งเตือนค่าเช่า (Rent Notifications) 🆕

สำหรับทรัพย์ที่ปล่อยเช่า:

- ระบบจะส่งแจ้งเตือนค่าเช่ารายเดือนอัตโนมัติผ่าน LINE และระบบแจ้งเตือนใน Dashboard
- **e-Signature Integration 📄:** สามารถส่งสัญญาให้ลูกค้าเซ็นออนไลน์ หรือเซ็นบนหน้าจอมือถือ (Sign-on-Device) ได้ทันที
- แจ้งเตือนก่อนสัญญาเช่าหมดอายุ (ตั้งค่าจำนวนวันล่วงหน้าได้)
- ดูรายการแจ้งเตือนทั้งหมดได้ที่หน้า **"Rent Notifications"**

---

## 7. การตั้งค่าโปรไฟล์และเว็บไซต์ (Settings)

- **LINE Integration:** เชื่อมต่อบัญชี LINE เพื่อรับการแจ้งเตือนงานใหม่
- **Social Integrations:** เชื่อมต่อ Facebook Page, Instagram Business, TikTok สำหรับ Social Posting
- **Social Post Template:** ตั้งค่ารูปแบบข้อความที่จะโพสต์อัตโนมัติ (ใช้ Dynamic Tags เช่น `{{title}}`, `{{price}}`)
- **Keyword Automation:** ตั้งค่า Comment-to-DM สำหรับ Facebook/Instagram
- **Site Settings:** สำหรับ Admin เท่านั้น สามารถเปลี่ยนโลโก้ และข้อมูลติดต่อบริษัทได้
- **AI Settings:** ตั้งค่า AI Model และ Parameters สำหรับระบบ AI ทั้งหมด

## 8. การจัดการทีมและพนักงาน (Team Management)

ฟีเจอร์สำหรับ **Admin** และ **Manager**:

- **ระบบสลับสาขา (Branch Switcher) 🆕:** สำหรับ Admin/Manager ที่ดูแลหลายสาขา สามารถคลิกชื่อสาขาที่แถบ Navbar ด้านข้าง เพื่อสลับไปดูข้อมูลของแต่ละสาขาได้ทันที
- **สร้างทีมใหม่:** ไปที่เมนู "Settings > Teams" เพื่อสร้างทีมและระบุหัวหน้าทีม (Manager)
- **จัดการสมาชิก:** สามารถย้ายเอเจนท์เข้าทีมหรือออกจากทีมได้ตลอดเวลา
- **โอนย้ายความรับผิดชอบ (Lead Transfer):** หากเอเจนท์ลาออก สามารถเลือกโอน Leads ทั้งหมดให้คนอื่นดูแลต่อได้ทันทีผ่านระบบ Bulk Action
- **Security Isolation:** ระบบจะคุมเข้มตามที่ Admin ตั้งค่าไว้ (เช่น เอเจนท์เห็นเฉพาะทรัพย์ของตนเอง)
- **Multi-Tenant Dashboard:** สำหรับ Admin/Manager ที่ดูแลหลายบริษัท สามารถคลิกชื่อบริษัทที่แถบ Navbar ด้านข้าง เพื่อสลับไปดูข้อมูลของแต่ละบริษัทได้ทันที
- **Security & Isolation:** ระบบคุมเข้มตามที่ admin ตั้งไว้
