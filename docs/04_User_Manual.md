# 👤 คู่มือการใช้งานสำหรับผู้ใช้ (User Manual)

วิธีใช้งานฟีเจอร์ต่างๆ ในระบบ Real Estate CRM สำหรับ Agent และ Admin

---

## 1. การจัดการทรัพย์สิน (Property Management)

- **เพิ่มทรัพย์สินใหม่:** กรอกข้อมูลเบื้องต้น ระบบจะมีปุ่ม **"AI Generator"** ช่วยเขียนคำบรรยายให้น่าสนใจ 3 ภาษา (ไทย, อังกฤษ, จีน) อัตโนมัติ
- **การจัดการรูปภาพ:** ลากวางรูปเพื่ออัพโหลด และสามารถเลือกรูป "Cover" ได้เพียงคลิกเดียว
- **สถานะทรัพย์สิน:** ปรับเปลี่ยนสถานะ (Active, Sold, Rented) เพื่อให้ระบบนำออกจากหน้าเว็บสาธารณะทันที

---

## 2. ระบบ AI Chatbot อัจฉริยะ (AI Search Assistant)

ลูกค้าสามารถพิมพ์คุยกับ Chatbot หน้าเว็บไซต์เพื่อหาบ้านที่ต้องการ:

- **Natural Language Search:** พิมพ์เช่น "หาคอนโดใกล้ BTS ราคากลางๆ" AI จะคำนวณและแสดงผลที่ตรงที่สุด
- **Lead Capture:** หากลูกค้าสนใจ AI จะขอชื่อและเบอร์ติดต่อ/LINE ID เพื่อสร้าง Lead เข้าสู่ระบบ CRM ให้อัตโนมัติ
- **Smart Matching:** ระบบจะคำนวณ Match Score ระหว่างความต้องการลูกค้ากับทรัพย์สินที่มีในระบบ

---

## 3. ระบบตอบกลับรวม (Omni-channel Inbox)

จัดการการพูดคุยกับลูกค้าทุกช่องทางในหน้าเดียว:

- **Centralized Chat:** ข้อความจาก LINE OA และช่องทางอื่นๆ จะมารวมกันที่หน้า Dashboard
- **Direct Reply:** Agent สามารถพิมพ์ตอบโต้ลูกค้าได้โดยตรงจากระบบ CRM ไม่ต้องสลับแอป
- **Notification:** เมื่อมีลูกค้าทักมาใหม่ ระบบจะส่งแจ้งเตือนเข้า LINE ของ Agent ทันที

---

## 4. การจัดการบทความ (Blog System)

1.  ไปที่เมนู **"Blogs"**
2.  ใช้ฟีเจอร์ **"AI Blog Writer"** เพื่อช่วยคิดหัวข้อและเนื้อหาที่น่าสนใจ (เช่น "5 เทคนิคเลือกคอนโดใกล้รถไฟฟ้า")
3.  การลงบทความบ่อยๆ จะช่วยให้เว็บไซต์ติดอันดับ Google (SEO) ได้ดีขึ้น

---

## 5. การตั้งค่าโปรไฟล์และเว็บไซต์ (Settings)

- **Line Integration:** เชื่อมต่อบัญชี LINE เพื่อรับการแจ้งเตือนงานใหม่
- **Site Settings:** สำหรับ Admin เท่านั้น สามารถเปลี่ยนโลโก้ และข้อมูลติดต่อบริษัทได้
- **Notification Prefs:** เลือกรับแจ้งเตือนเฉพาะเรื่องที่สำคัญต่อคุณ

---

## 6. การจัดการทีมและพนักงาน (Team Management)

ฟีเจอร์สำหรับ **Admin** และ **Manager**:

- **สร้างทีมใหม่:** ไปที่เมนู "Settings > Teams" เพื่อสร้างทีมและระบุหัวหน้าทีม (Manager)
- **จัดการสมาชิก:** สามารถย้ายเอเจนท์เข้าทีมหรือออกจากทีมได้ตลอดเวลา
- **โอนย้ายความรับผิดชอบ (Lead Transfer):** หากเอเจนท์ลาออก สามารถเลือกโอน Leads ทั้งหมดให้คนอื่นดูแลต่อได้ทันทีผ่านระบบ Bulk Action
- **Security Isolation:** ระบบจะคุมเข้มตามที่ Admin ตั้งค่าไว้ (เช่น เอเจนท์เห็นเฉพาะทรัพย์ของตนเอง)
