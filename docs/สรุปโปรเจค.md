## 1. ภาพรวมโปรเจค

### 1.1 คำอธิบายโปรเจค

**VC Connect Asset CRM** เป็นระบบจัดการธุรกิจอสังหาริมทรัพย์แบบครบวงจร ออกแบบมาเพื่อ:

- นายหน้าอสังหาริมทรัพย์
- บริษัทตัวแทนอสังหาฯ
- นักพัฒนาอสังหาริมทรัพย์
- ทีมขายอสังหาฯ

### 1.2 จุดเด่นของระบบ

| จุดเด่น                   | รายละเอียด                                                                     |
| ------------------------- | ------------------------------------------------------------------------------ |
| ✅ **AI-First CRM**       | Chatbot ค้นหาทรัพย์ และ AI เจนเนื้อหา 3 ภาษา                                   |
| ✅ **Omni-channel**       | รวมแชท LINE / FB / IG / WhatsApp เข้าที่เดียว                                  |
| ✅ **Social Media**       | โพสต์ทรัพย์ลง FB/IG/TikTok + Keyword Automation                                |
| ✅ **Agent Isolation**    | ระบบแยกฐานข้อมูลทรัพย์สิน/ลูกค้า ป้องกันข้อมูลรั่ว                             |
| ✅ **AI Valuation 🆕**    | ระบบ AVM ประเมินราคาทรัพย์อัจฉริยะ (Max Profit/Market/Quick Sale)              |
| ✅ **Smart Contracts 🆕** | อัปโหลด Word Template เติมข้อมูล Lead ลง PDF พร้อมเซ็นออนไลน์ (E-Sign)         |
| ✅ **Team & Managers**    | โครงสร้างองค์กร จัดทีมและหัวหน้าทีม พร้อมระบบ Multiple Branches / Multi-Tenant |
| ✅ **Modern Stack**       | Next.js 16, React 19, Tailwind CSS 4                                           |
| ✅ **Multi-language**     | รองรับ 3 ภาษาเต็มรูปแบบ (TH, EN, CN)                                           |
| ✅ **Security & Audit**   | ระบบบันทึก Audit Logs, RLS, Cookie Consent (PDPA)                              |
| ✅ **SEO-Ready**          | Sitemap, Meta tags, Structured Data                                            |
| ✅ **Cloud-Native**       | Supabase (PostgreSQL + Auth + Storage)                                         |

### 2.1 🏠 ระบบจัดการทรัพย์สิน (Property Management)

```
📁 features/properties/
├── actions.ts              # Server Actions สำหรับ CRUD
├── components/             # UI Components
│   ├── PropertyForm/       # ฟอร์มเพิ่ม/แก้ไขทรัพย์ (6 Steps)

│   └── PropertyTable/      # ตารางรายการทรัพย์
```

- **Turn-key Package:** เสนอขายเป็นระบบสำเร็จรูปพร้อม Source Code และเอกสารครบชุด
