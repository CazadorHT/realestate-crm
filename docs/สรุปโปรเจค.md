## 1. ภาพรวมโปรเจค

### 1.1 คำอธิบายโปรเจค

**Real Estate CRM** เป็นระบบจัดการธุรกิจอสังหาริมทรัพย์แบบครบวงจร ออกแบบมาเพื่อ:

- นายหน้าอสังหาริมทรัพย์
- บริษัทตัวแทนอสังหาฯ
- นักพัฒนาอสังหาริมทรัพย์
- ทีมขายอสังหาฯ

### 1.2 จุดเด่นของระบบ

| จุดเด่น                 | รายละเอียด                             |
| ----------------------- | -------------------------------------- |
| ✅ **All-in-One**       | CRM + Website + Blog ในระบบเดียว       |
| ✅ **AI-Powered**       | ใช้ AI ช่วยเขียนคำบรรยาย/บทความ        |
| ✅ **LINE Integration** | แจ้งเตือนอัตโนมัติผ่าน LINE            |
| ✅ **Multi-language**   | รองรับไทย/อังกฤษ (พร้อมขยายจีน)        |
| ✅ **SEO-Ready**        | Sitemap, Meta tags, Structured Data    |
| ✅ **Modern Stack**     | Next.js 16, React 19, Tailwind CSS 4   |
| ✅ **Cloud-Native**     | Supabase (PostgreSQL + Auth + Storage) |

## 2. รายละเอียดฟีเจอร์

### 2.1 🏠 ระบบจัดการทรัพย์สิน (Property Management)

```
📁 features/properties/
├── actions.ts              # Server Actions สำหรับ CRUD
├── components/             # UI Components
│   ├── PropertyForm/       # ฟอร์มเพิ่ม/แก้ไขทรัพย์ (6 Steps)
│   ├── PropertyCard/       # Card แสดงทรัพย์
│   └── PropertyTable/      # ตารางรายการทรัพย์
└── hooks/                  # Custom Hooks
```

**ความสามารถ:**

- ✅ เพิ่ม/แก้ไข/ลบ ทรัพย์สิน
- ✅ รองรับประเภทหลากหลาย: บ้าน, คอนโด, ทาวน์โฮม, ที่ดิน, อาคารพาณิชย์, โกดัง, สำนักงาน
- ✅ อัปโหลดรูปภาพหลายรูป พร้อม Drag & Drop
- ✅ Smart Match (จับคู่ลูกค้ากับทรัพย์)
- ✅ AI Description Generator (สร้างคำบรรยายอัตโนมัติ)
- ✅ ระบบสถานะ: Draft, Active, Archived, Under offer, Reserved, Sold, Rented
- ✅ SEO Slug อัตโนมัติ
- ✅ ระบบ Co-Agent (นายหน้าร่วม)
- ✅ ข้อมูลเจ้าของทรัพย์ (Owner)
- ✅ ฟีเจอร์พิเศษ: วิว, สิ่งอำนวยความสะดวก, สถานที่ใกล้เคียง
